def AbstractItem.AbstractItem.__str__ (   self)

Returns a representation of the state of this object as a multiline string.

The format is: 

    type
    title
    (genre)
    status

The status is either 

    Rented: yyyy-mm-dd

or 

    Available

where "yyyy-mm-dd" is the current due date.
